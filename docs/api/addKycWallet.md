# Add Kyc Wallet

**Endpoint**: `/addKycWallet`

**HTTP Method**: `POST`

**Params**: 
``` json
{
    "id": "id",
    "wallet": "0xd198...ce351",
    "signature": "signature"
}
```

## Description

This endpoint allows you to add a KYC (Know Your Customer) wallet.
