# Insert Pool

**Endpoint**: `/insertPool`

**HTTP Method**: `POST`

**Params**: 
``` json
{
  "poolId": "",
  "name": "",
  "description": "",
  "image": "",
  "website": "",
  "twitter": "",
  "collateralTokens": [],
  "chain": ""
}
```

## Description

The request should include a payload in the raw request body type with parameters such as poolId, name, description, image, website, twitter, collateralTokens, and chain.

## Response
